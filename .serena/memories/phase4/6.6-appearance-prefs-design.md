## Locked decisions (user-approved via AskUserQuestion)
1. **Accent picker = curated swatches** (NOT free color wheel). ~6 tuned oklch presets `{l,c,h}`: Terracotta (default = current `0.66 0.18 38`), Blue, Green, Purple, Magenta, Amber. `--acc-fg` stays `#fff`.
2. **Sidebar style = width tiers** via `data-sidebar` attr on documentElement, pure CSS on `.ss-side`: rail (icon-only ~64px, labels hidden, title tooltips), compact (current default), wide (roomier padding/hit targets).
3. **Step badges toggle = CSS-only** (pending final confirm): `data-step-badges="0|1"` on documentElement hides `.ss-nav-num` (sidebar) AND `.step-pill` (PageHead). Badges always render in DOM; CSS hides them. Avoids prop-drilling through Sidebar/PageHead → existing tests stay green. (Sidebar already has a `showSteps` prop + `ss-nav-num`; PageHead renders `.step-pill`.)
